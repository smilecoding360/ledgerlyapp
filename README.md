# ledgerlyapp.github.io
Ledgerly app webpage

## Project Structure

- `index.html` - landing page markup
- `css/styles.css` - active page styles
- `css/legacy/styles.legacy.css` - previous unused stylesheet kept for reference
- `js/scripts.js` - carousel and interaction logic
- `assets/favicon.svg` - SVG favicon
- `assets/images/branding/` - app icon images
- `assets/images/badges/` - store badge assets
- `assets/images/screenshots/` - app screenshot images
- `assets/images/backgrounds/` - decorative background assets
- `docs/app-ads.txt` - GitHub Pages app-ads.txt copy

If your GitHub Pages site is published from the `docs/` folder, the static site files must be placed inside `docs/`.

The `app-ads.txt` file must be accessible at `https://<your-domain>/app-ads.txt`. For a GitHub Pages setup published from `docs/`, keep the file at:
- `docs/app-ads.txt`
